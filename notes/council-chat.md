Interaction Logic
Mode 1 — All

(Default)

ALL

means:

Athena orchestrates dynamically.

The system decides:

which agents are relevant
in what order
how much weight each gets

This is the best default experience.

Example:

Question:

“What’s happening with AI stocks?”

System invokes:

Sigma
Janus
Achilles
Athena

Maybe NOT Atlas if unnecessary.

Mode 2 — Manual Selection

User manually enables:

Atlas
Achilles

Now ONLY those agents participate.

This is VERY powerful.

Because users can intentionally shape cognition.

Examples:

Macro-focused user
Atlas + Janus
Trader
Sigma + Achilles
Allocator
Atlas + Athena + Achilles

invoking intelligences.

relevance
